结构化存储PHP SDK
========================
1. 安装第三方依赖

```
./composer.phar install
```

2. 运行示例代码，测试是否正常(需要修改示例代码中的AppID/AppSecret)

```
php examples/Basic.php
```


SDS PHP SDK User Guide
========================

```

```
